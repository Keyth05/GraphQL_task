# GraphQL Hello World 🚀

This is a simple **GraphQL** implementation using **Node.js** and **Apollo Server**. The purpose of this project is to set up a basic GraphQL server that returns a "Hello World" message.

## Requirements 🛠️

Before you get started, make sure you have the following installed on your system:

- **Node.js** (v14 or higher)
- **npm** (Node.js package manager)
